# **多人聊天室项目**

## **项目介绍**：

该项目的核心功能为让用户使用浏览器访问主界面进行多用户实时通讯。

---


## **多人聊天室项目具体功能**：

- 登录注册
- 添加好友
- 私聊
- 群聊
- 文本传输
- 图片、小文件传输
- 表情包

---

**前端语言**：JavaScript, HTML, CSS  
**后端语言**：JavaScript(node.js)  
**客户端**：Microsoft Edge浏览器  
**服务器**：阿里云服务器  
**数据库**：MySql  
**数据传输**：Json数据格式

---
